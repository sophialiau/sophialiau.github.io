Technical Overview

This portfolio website is built using HTML, CSS, and JavaScript, ensuring a responsive and accessible design. The layout is structured to provide a clear and intuitive navigation experience, with sections dedicated to my projects, skills, and contact information. I created the custom layout on Figma, which was then personally translated into code using HTML and CSS. The website features a clean and modern aesthetic, with a focus on typography and visual hierarchy to enhance readability and user engagement. Most images and graphics used in the project are created by me, ensuring a cohesive and personalized design language throughout the site. The design incorporates modern web standards and best practices to ensure compatibility across various devices and browsers. The website is hosted on Vercel, allowing for easy updates and version control.
